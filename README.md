# 💰 Mini Proyecto 4 — Clasificación de Préstamos con Random Forest en PySpark

## Descripción | Description

**ES:** Este proyecto aplica **Random Forest con PySpark** para clasificar solicitudes de préstamos según su riesgo de incumplimiento. Se trabajó con procesamiento distribuido de datos, ingeniería de características y evaluación del modelo en un entorno de Big Data.

*This project applies **Random Forest with PySpark** to classify loan applications based on their default risk. Distributed data processing, feature engineering, and model evaluation in a Big Data environment were performed.*
